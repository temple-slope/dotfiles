# クリップボード出力

Claude Codeのターミナル描画は自動的にインデントを付与するため、テキストをそのままコピーすると不要なインデントが含まれる。このスキルはテキストを `pbcopy` 経由でクリップボードに直接格納し、インデントなしで貼り付け可能にする。

## 手順

1. ユーザーの依頼に基づきテキストを生成する
2. `printf '%s'` と `pbcopy` を使ってクリップボードに格納する
3. ファイル出力は不要（クリップボード直接）

## 実行方法

```bash
printf '%s' 'ここにテキスト内容' | pbcopy
```

### 注意事項

- テキスト内にシングルクォートが含まれる場合はヒアドキュメントを使う:

```bash
pbcopy <<'EOF'
テキスト内容
EOF
```

- macOS の `pbcopy` を使用する（Linux の場合は `xclip -selection clipboard`）
- 完了後「クリップボードに入れました」と報告する
- `[メールアドレス]` 等の差し替え箇所がある場合はその旨伝える
